# Smart Door Alarm System

An Arduino-based security system that detects door opening and activates an alarm.

## Components Required
- Arduino Uno / Nano
- Magnetic Reed Switch
- Active Buzzer
- LED
- 220 Ohm Resistor
- Breadboard
- Jumper Wires
- USB Cable

## Features
- Detects door open/close
- Audible alarm
- LED status indicator
- Low-cost and simple design

## Working Principle
When the door opens, the reed switch changes state. Arduino senses this and activates the buzzer and LED.

## Pin Connections
Reed Switch  -> D2  
Buzzer (+)   -> D8  
LED (+)      -> D7  
All GNDs     -> GND

## Applications
- Home security
- Office doors
- Lockers
- Cabinets
